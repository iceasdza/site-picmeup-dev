PICMEUP(web application)

เครื่องมือในการพัฒนา
NodeJS  
React 
Download NodeJS https://nodejs.org/en/download/current/
และติดตั้งตัว NodeJS เพื่อใช้คำสั่ง npm
 เปิด cmd ใช้คำสั่ง npm -v เพื่อตรวจสอบว่าติดตั้งสำเร็จหรือไม่
ใช้คำสั่ง npm install -s เพื่อทำการติดตั้งตัว Modules ของตัว project
เมื่อ install เสร็จใช้คำสั่ง npm start เพื่อทำการ run project
*** ต้องทำการ run ตัวของทาง API ก่อน
